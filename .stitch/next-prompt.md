Design and refine the ReservePro Dashboard Workspace to support detailed room calendars, multi-day bookings, and a list of active users.

**DESIGN SYSTEM (REQUIRED):**
* **Atmosphere:** Clean, modern, highly organized, productivity-focused. Inspired by Ant Design v5.
* **Colors:** Primary Blue (#1677ff), Secondary Slate (#595959), Background Light Gray (#f7f9fc), Container White (#ffffff), Border Light (#f0f0f0).
* **Typography:** Inter font, Semibold headers.
* **Components:** Buttons have 6px rounded corners, middle height is 40px, primary buttons have solid blue background (#1677ff) and white text. Inputs have 6px rounded corners.

**Page Structure:**
1. Global Header with ReservePro logo, brand title, and User Profile.
2. Sidebar on the left listing Rooms with capacities.
3. Selected Room banner with active state indicator and room features.
4. Bookings Table displaying host/guest names, start time, end time, and cancel/delete actions.
5. Create Booking form with Guest Name input, Start/End date-time selectors, and a confirmation button.
